# Tennessee Eastman (TE) — 控制回路對照

> 目的：給 ME 快速定位「被控量 ↔ 操縱量 ↔ 感測／致動 ↔ 控制目標」；亦可作為 DS/DE 的對齊表。

| 回路編號 | 被控量 (CV)             | 操縱量 (MV)                          | 感測/致動元件 (例)                 | 控制目標與邏輯                                                     | 常見權衡/風險                             |
|---------|--------------------------|--------------------------------------|------------------------------------|--------------------------------------------------------------------|------------------------------------------|
| CL-01   | 反應器溫度               | 冷卻介質流量 / 夾套閥                | TT/TIC → FCV                       | 維持反應溫度，抑制放熱峰值；前饋可用投料/熱負荷估計                 | 過冷影響轉換率；冷卻失效→安全風險          |
| CL-02   | 反應器壓力               | 頂部壓力控制閥 / 排氣                | PT/PIC → PCV                       | 穩壓避免超壓；與分離槽/回收線協調                                   | 排氣過大影響收率；關太多壓力上升           |
| CL-03   | 反應器液位               | 反應器出料流量                       | LT/LIC → FCV                       | 抑制液位擾動，防止乾塔/外溢                                         | 出料調整影響下游負荷                      |
| CL-04   | 分離槽壓力               | 冷凝器負荷 / 頂部閥                  | PT/PIC → TC/PCV                    | 維持壓力與冷凝率，搭配塔/回收線                                     | 過度冷凝→液位波動                         |
| CL-05   | 分離槽液位               | 底部液相流量                         | LT/LIC → FCV                       | 穩定供料至下游汽提塔                                               | 液位低導致汽提塔斷料                      |
| CL-06   | 回收流量/壓差            | 壓縮機轉速/導葉或回收閥              | FT/DP → FIC/PIC → VSD/FCV          | 支撐循環品質與反應環境穩定                                         | 喘振風險；能耗上升                        |
| CL-07   | 汽提塔塔壓               | 頂部冷凝/頂部閥                      | PT/PIC → TC/PCV                    | 維持塔壓與分離驅動力                                               | 塔壓過高→安全；過低→冷凝困難              |
| CL-08   | 汽提塔塔底液位           | 塔底出料流量                         | LT/LIC → FCV                       | 保持穩定產品流與再沸器浸沒度                                       | 液位波動影響產品品質                      |
| CL-09   | 產品規格（代理：塔頂/底T）| 回流比 / 再沸器熱負荷                | TT/TIC → FCV/TC                    | 以溫度代理組成做品質控制（輔以分析儀更佳）                           | 代理偏差；負荷限制                         |
| CL-10   | 惰性/不凝累積（代理）    | 排放流量（purge）                    | FT/FIC → PCV                       | 以最小 purge 控制組成與循環品質                                     | 排放↑成本↑；排放↓→惰性累積                |
| CL-11   | 投料比（Feed Ratio）     | 各原料流量（主控＋比率從控）          | FT/FIC (master/slave)              | 以比率鎖維持化學計量與選擇性                                       | 比率漂移影響收率與副反應                  |
| CL-12   | 公用工程供應（代理）      | 冷卻水閥/蒸汽閥                      | PT/TT/FT → 控制閥                  | 穩定 utility 供應，降低耦合擾動                                     | 公用波動傳遞到主製程                      |

> 對齊位（待填）：

> - CL-06 對應 tag：FIC-[回收線] → VSD/FCV-[壓縮機/回收閥]
